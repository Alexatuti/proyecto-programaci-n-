# Sistema de Gestión de Notas e Investigaciones en C++

## Descripción
Este proyecto consiste en un sistema desarrollado en lenguaje C++ que permite gestionar
notas académicas e investigaciones de estudiantes y docentes. El sistema funciona mediante
menús interactivos en consola y utiliza archivos CSV para el almacenamiento de la información.

## Funcionalidades
- Registro de notas académicas
- Visualización de notas
- Búsqueda de notas por distintos criterios
- Ordenamiento de notas
- Actualización y eliminación de registros
- Gestión de investigaciones universitarias

## Tecnologías Utilizadas
- Lenguaje: C++
- Librerías estándar de C++
- Archivos CSV
- GitHub para control de versiones

## Estructura del Proyecto
- main.cpp : código fuente principal
- notas.csv : archivo de almacenamiento de notas
- investigaciones.csv : archivo de almacenamiento de investigaciones

## Ejecución del Programa
1. Compilar el archivo `main.cpp`
2. Ejecutar el programa desde la consola
3. Utilizar el menú para acceder a las funcionalidades
